Deskripsi Proyek:
> Aplikasi ini merupakan sistem pencatatan data anime yang dapat digunakan untuk menambah, menampilkan, mengubah, dan menghapus informasi anime. Fitur yang tersedia meliputi pencarian anime, penampilan detail lengkap anime (judul, skor, status total episode, tanggal rilis, produser, studio, genre, dan sinopsis). Selain itu, tiap anime juga diberi cover agar lebih informatif dan menarik.

Alasan Memilih Tema:  
> Saya memilih tema ini karena saya kepikirannya tema ini. Anime sendiri memiliki komunitas yang cukup masif dan selalu memiliki judul baru tiap tahunnya. Database anime juga sangat besar, terdapat ribuan judul yang dapat diinput. Jadi, saya cukup yakin tema ini akan tetap relevan untuk jangka waktu yang lama.
